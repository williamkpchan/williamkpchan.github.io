Disable the Google Chrome Software Reporter Tool using file permissions


%localappdata%\  = C:\Users\User\AppData\Local\

Go to %localappdata%\google\chrome\User Data\SwReporter\39.195.200.3\
Right-click software_reporter_tool.exe and select Properties

select Deny in the permissions list for all users.
This will make sure the tool is not allowed to run automatically.
