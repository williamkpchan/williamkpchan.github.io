第一部分：MIDI基础概念

什么是MIDI？

MIDI（Musical Instrument Digital Interface）是乐器间的"通用语言"，传递的是演奏指令（音高、力度、时值等），而非实际音频。

核心要点：

  • MIDI信号 ≠ 声音，需要连接音源才能发声
  • 三个关键接口：MIDI OUT（输出）、MIDI IN（接收）、MIDI THRU（转发）
  • 现代设备多用USB连接，传统设备用5针DIN接口

快速入门工具推荐

MidiEditor 是一款免费的网页版MIDI编辑器，无需下载，打开浏览器就能用。

三步开始创作：

  1. 连接设备：设置 → MIDI I/O → 勾选USB MIDI键盘端口
  2. 实时录制：点击红色录制按钮，开始演奏
  3. 多轨编辑：钢琴卷帘窗中调整音符、力度、时长

实用技巧：

  • 按 Q 键量化音符，自动对齐节奏
  • 支持128种标准GM音色库切换
  • 导出为标准.mid文件，兼容所有主流音乐软件

第二部分：音色选择与搭配

什么是MIDI音色？

MIDI音色（Timbre）是声音的个性与质感。同样的C音，钢琴和小提琴演奏出来感觉完全不同——这就是音色的魔力。

在MIDI中，音色不是音频文件，而是一组控制音源的指令参数。当你按下琴键，MIDI信号告诉音源："用0号音色（大钢琴）以100的力度演奏C4音"。

GM标准：128种通用音色

GM（General MIDI） 是1991年确立的通用标准，规定了128种音色的统一编号，确保不同设备播放MIDI文件时音色一致。

音色的三种技术实现

1. 合成器（Synthesizer）

通过算法凭空"制造"声音波形。优势是能创造出自然界不存在的声音，如电子音色、科幻音效。

常见合成方法：

  • 减法合成：从复杂波形中滤除频率（模拟合成器经典做法）
  • FM合成：用频率调制产生金属感、铃声质感（如Yamaha DX7）
  • 波表合成：预存数字波形，成本低，常见于声卡

2. 采样器（Sampler）

录制真实乐器演奏，分配到MIDI键盘上。弹C键就播放录好的C音录音，音色最真实。

主流采样器软件：

  • KONTAKT：行业标准，支持海量音色库
  • SampleTank：操作简便，适合入门
  • DirectWave：FL Studio自带，支持16通道多音色

3. SoundFont/DLS

一种"波形表"格式，将采样音色打包成文件。播放器加载后就能作为音源使用。

黄金搭配公式

风格 核心组合 效果
流行歌曲 0号钢琴 + 25号钢弦吉他 + 33号贝司 + 40号小提琴 温暖饱满
史诗配乐 14号管钟 + 48号弦乐合奏 + 60号圆号 + 86号铜管 宏大辉煌

Lofi治愈 4号Rhodes电钢琴 + 10号八音盒 + 75号排箫 复古梦幻

四大搭配原则

  1. 频率互补
    • 低频：贝司、大鼓（30-250Hz）
    • 中频：吉他、钢琴（250-4000Hz）
    • 高频：小提琴、镲片（4000Hz以上）
    • 避免同频段乐器堆叠，否则声音浑浊
  2. 分清主次
    • 主奏乐器音量突出，铺底乐器降低音量或放左右声道
    • 用声像（Pan）分离：主弦乐放左，钢琴放右
  3. 风格统一
    • 民谣：木吉他、口琴、手鼓
    • 摇滚：失真吉他、架子鼓、合成贝司
    • 古风：古筝、二胡、RMX鼓机
  4. 情绪匹配
    • 宁静夜晚 → 柔和弦乐、风声（避免尖锐音色）
    • 紧张悬疑 → 低频合成器、不规则打击乐

常见误区

误区 正确做法
音色越多越好 3-5种精心搭配的音色 > 10种堆砌
只用默认钢琴 根据风格选择：流行用明亮钢琴，爵士用柔和电钢琴
忽视MIDI控制器 用1号颤音轮、11号表情控制器增加动态
忽略音色品质 同样0号钢琴，专业音源 vs 廉价声卡差距巨大

第三部分：量化技巧详解

什么是量化？

量化是将演奏或输入的音符自动对齐到节拍网格的过程。就像磁铁把铁屑吸附到网格线上一样，量化把"差不多"在节拍上的音符变成"精确"在节拍上。

为什么需要量化？

  • 真人演奏难免有微小偏差（提前或延后几毫秒）
  • 多个乐器节奏不一致时，整体听起来会"散"
  • 电子音乐需要精确的机械感节奏

量化三要素

1. 量化精度（Grid/Note Value）

决定音符对齐到多细的格子：

精度值 名称 适用场景
1/4 四分音符 慢速和弦、氛围音乐
1/8 八分音符 流行歌曲、一般节奏
1/16 十六分音符 最常用，适合大多数风格
1/32 三十二分音符 快速电子乐、复杂琶音
1/8T 八分音符三连音 爵士、布鲁斯、华尔兹
1/16T 十六分音符三连音 复杂的切分节奏

选择原则：看片段中最短的音符是什么，就选对应的精度。比如有十六分音符的加花，就必须选1/16，否则会丢失细节。

2. 量化强度（Strength/Amount）

控制"磁力"有多强，这是人性化演奏的关键：

  • 100%：音符完全对齐网格（机械、精确、适合电子舞曲）
  • 70-80%：轻微修正，保留自然感（推荐用于钢琴、吉他）
  • 50%：一半修正，明显的"慵懒"或"推进"感
  • 0%：不量化

实战建议：

  • 鼓组：90-100%（节奏核心必须稳）
  • 贝斯：80-90%（与鼓紧密配合）
  • 和弦：60-70%（保留呼吸感）
  • 旋律：50-70%（根据风格调整）

3. 摇摆（Swing）

让偶数拍（如第2、4个十六分音符）延后，产生摇摆感：

  • 50%：均匀音符，无摇摆（标准电子节拍）
  • 54-58%：轻微摇摆（House、Funk常用）
  • 60-66%：明显摇摆（爵士、Hip-hop）
  • 75%：最大摇摆（复古摇摆乐）

应用技巧：在Ableton Live中，可以给踩镲（Hi-hat）加54%的摇摆，而底鼓和军鼓保持50%，这样既有律动又不散乱。

各DAW量化操作速查

Ableton Live

  • 快速量化：选中音符 → Ctrl/Cmd + U（按当前网格）
  • 高级设置：Ctrl/Cmd + Shift + U（调整强度、摇摆、是否量化音符尾）
  • 录音时自动量化：编辑菜单 → 录音量化 → 选择精度

FL Studio

  • 快速量化：钢琴卷帘窗 → 快捷工具菜单 → Quick quantize（音符头尾都对齐）
  • 仅量化起始时间：Quick quantize start times（保留原有时值）
  • 吸附设置：钢琴卷帘窗左上角选择网格精度（Beat/Step等）

Logic Pro

  • 编辑器内量化：选中音符 → 左侧检查器 → 选择量化精度（1/16、1/8等）
  • 轨道区域量化：左侧轨道头 → MIDI Region → 量化
  • 人性化：量化后使用"人性化"功能添加随机偏移

Cubase

  • 选中MIDI数据 → 按 Q 键（默认设置）
  • 详细设置：MIDI菜单 → 量化设置 → 调整摇摆、吸附区域、随机量化等

实战案例：Deep House鼓组量化

假设你录了一段鼓组，但拍手声（Clap）有点飘忽：

  1. 修复拍手节奏：
    • 选中拍手MIDI片段的所有音符
    • 右键 → 量化设置
    • 精度：1/16（因为有十六分音符加花）
    • 强度：70%（保留轻微人性化）
    • 应用 → 拍手现在稳了，但不死板
  2. 给踩镲加摇摆：
    • 选中踩镲片段
    • 从浏览器拖入 Swing SP 1200 16ths 54 律动模板
    • 强度设为 70%
    • 听感：从机械变得有Funk律动
  3. 整理贝斯：
    • 贝斯必须与底鼓（Kick）紧密咬合
    • 强度设为 100%
    • 勾选"量化音符结尾"（避免低音重叠浑浊）

量化避坑指南

误区 后果 正确做法
全部100%量化 音乐像机器人演奏，毫无生气 不同乐器用不同强度
精度选太粗 快速音符被错误对齐，节奏变形 选片段中最短音符的精度
忽视三连音 爵士/布鲁斯的摇曳感消失 遇到三连音时切换1/8T或1/16T
量化音符长度 长音被截断，短音被拉长 通常只量化起始位置
忘记撤销 原始演奏感永久丢失 量化前复制一轨备份

进阶技巧

  • 随机量化（Random Quantize）：在Cubase等DAW中，可以给量化添加随机偏移，让完美的网格音符有微妙的"人味"
  • 冻结量化（Freeze Quantize）：Logic Pro等软件支持冻结某些音符不被量化，适合保留特定装饰音的即兴感
  • 律动模板（Groove Pool）：Ableton Live的Groove Pool可以将经典鼓机的摇摆（如Akai MPC的60%摇摆）应用到你的MIDI上，比单纯数字更有"味道"
  • 量化力度（Velocity Quantize）：除了时间，还可以量化音符力度（音量）。比如让所有军鼓的力度统一为100，或添加随机变化

MidiEditor 定位分析

MidiEditor 是一款轻量级、开源、跨平台的MIDI编辑器，适合快速编辑和简单创作。但它的功能相对基础，主要适合：

  • 快速修改MIDI文件

局限性：缺乏专业音源、混音功能、复杂自动化等现代音乐制作必备功能。

路径A：AI直接生成（推荐新手）

Suno AI —— 目前最强

流程：

  1. 访问 suno.com
  2. 选择 "Custom Mode"（自定义模式）
  3. 粘贴你的歌词到歌词框
  4. 输入风格提示（如"Pop, Female Vocal, Piano"）
  5. 生成后进入 Suno Studio（付费功能）
  6. 右键音轨 → "Get MIDI" 导出MIDI文件

优势：

  • 支持中文歌词，人声质量极高
  • 可提取分轨MIDI（鼓、贝斯、钢琴等）
  • 2025年V5版本支持MIDI导出功能

局限：MIDI导出需要Premier付费计划（约¥150/月）

Udio —— 情感表达更强

流程：

  1. 访问 udio.com
  2. 选择 "Custom Write your own lyrics"
  3. 分段输入歌词（每次约33秒）
  4. 使用 Extend 功能延长歌曲
  5. 下载音频 → 用Klangio等工具转MIDI（见下文）

优势：

  • 人声情感更真实，呼吸感强
  • 免费版每月可生成1200首
  • 支持多种语言

局限：暂时不直接支持MIDI导出，需音频转MIDI

  4. 导出MIDI到FL Studio等DAW继续编辑

路径C：专业DAW workflow

FL Studio —— 嘻哈/电子首选

歌词辅助作曲流程：

  1. 在 钢琴卷帘窗 中输入歌词标记（使用"Stamp"功能）

Ableton Live —— 现场/实验音乐

独特功能：

  • Convert Audio to MIDI：哼唱旋律 → 自动转MIDI

  • Convert Drums to MIDI：Beatbox节奏 → 鼓组MIDI

歌词工作流程：

关键注意事项

  1. 中文歌词处理：Suno对中文支持最好，Udio次之，其他工具可能需要拼音或英文中转
  2. MIDI质量：AI提取的MIDI可能需要量化（见之前的教程）和力度调整才能使用

总结

核心要点回顾

  • MIDI基础：MIDI是控制指令而非声音，需要音源才能发声
  • 音色选择：遵循频率互补、分清主次、风格统一、情绪匹配四大原则
  • 量化技巧：量化 = 选择精度 + 调整强度 + 适量摇摆
  • 工具选择：MidiEditor是"记事本"，FL Studio/Ableton/Logic是"Office套件"
  • AI歌词生成：Suno直接生成MIDI最方便，Udio情感更丰富，Klangio是音频转MIDI神器

新手最快上手路径

  1. 先用 0号大钢琴 写旋律，33号贝司 写根音
  2. 加入 48号弦乐合奏 铺底，25号吉他 分解和弦
  3. 用 81号锯齿波 做副歌Lead音色提亮
  4. 用 Suno 生成完整歌曲，导出MIDI到 FL Studio 精修
  5. 用 70%强度量化 修正节奏，54%摇摆 增加律动

整个过程30分钟即可完成第一首作品！
