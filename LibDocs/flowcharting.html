LucidChart

1 — LucidChart

Online collaboration tool with an unlimited free account that allows creation of an unlimited number of diagrams with a limit of 60 objects per diagram.

  • Creates flowcharts, organizational charts, website wireframes, UML (Unified Modeling Language) designs, mind maps, software prototypes, and diagram types;
  • Web app that works in Google Chrome, Firefox, Safari, Microsoft Edge, and Internet Explorer 8;
  • Cloud-based;
  • Offers collaboration between teams in real time;
  • Allows publication and sharing;
  • Exports to vector PDFs, JPEG, and PNG images;
  • Imports from Visio, OmniGraffle 6, and OmniGraffle 6 Pro;
  • Integrates with Google Apps, Confluence, and JIRA;
  • Features Enterprise-level security and admins controls.


Favorite feature
  • It’s a web app, and it also works in tablets.



Microsoft Visio

2 — Microsoft Visio

Diagramming and vector graphics application; part of Microsoft Office family.

Pros
  • Creates flowcharts, organizational charts, website wireframes, UML designs, mind maps, software prototypes, and diagram types;
  • Allows publication and sharing;
  • Features Enterprise-level security and admins controls
  • Not a cloud and web app;
  • Doesn’t allow collaboration between teams in real time;
  • Has to be downloaded and installed;
  • Windows only.


Favorite feature
  • Integration with Microsoft Excel, SQL server, SQL Azure, and SharePoint.



Creately

OmniGraffle

6 — OmniGraffle

Graphical user interface mockup and website wireframe builder application available for Mac, iPhone, and iPad.

Pros
  • Creates flowcharts, organizational charts, website wireframes, UML (Unified Modeling Language) designs, mind maps, software prototypes, and diagram types;
  • Not a cloud-based or a web-app;
