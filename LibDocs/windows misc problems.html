microsoft software protection

Disable Microsoft Software Protection Service sppsvc without losing Office and Windows Activation
created a batch comand file with just one line that can help with this problem (you can also run the command prompt as an admin user and type it manually):
Code:
net stop sppsvc
If you need the service again, for any reason, you can try
Code:
net start sppsvc
I know it's something basic, but can help specially with lower end machines using a pirate version of Windows and Office.

windows 10 startup folder

windows-10-startup-folder
C:\Users\User\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
The All Users Startup Folder:
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp
The Current User Startup Folder:
C:\Users\[Username]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

quick access All Users Startup Folder in Windows 10:
Windows Key + R, type shell:common startup

quick access Current User Startup Folder:
Windows Key + R, type shell:startup.

windows search

依照微軟官方網站提供說明文件指出如果關閉Windows Search 有可能造成部分功能無法正常運作(只是被關閉的意思)
依照許多的客製化調整我們必須做出是當的設定值。
在此我選擇關閉Windows Search，以下兩種做法提供大家參考。
01.開啟你的service.msc找到Windows Search此項服務，並將此服務停用。
02.開啟控制台找到開啟或關閉Windows功能，內有Windows Search此項選項，並將此選項取消勾選即可。
以上都是可以關閉服務的方法，但是其中卻有優缺點
01.此項辦法個人較於偏愛，因為只是關閉部分的自動化搜索並可讓電腦較為順暢。
關閉完整系統搜索服務當哪天若有需求要再使用，需再次開啟或關閉極為麻煩並且也不是每個人都會設定此類相關，即使在電話中教學還是有部分的困難存在。
