音符（note）

在樂譜中，不同種類的音符是在表示不同的相對時間長度。 在大部分的樂譜中，常常會使用到的 最長的音符是空心的全音符（whole note）； 加了一條直線的空心音符叫做二分音符（half note），是全音符長度的一半； 帶有直線的實心音符叫做四分音符（quarter note），是二分音符長度的一半； 長度再減半的是帶有尾巴的八分音符（eighth note）； 然後是帶有兩條尾巴的十六分音符（sixteenth note）。 最常見的五種音符 從這些音符的英文名稱可以很明顯地看出，我們在中文講的「二分、四分、八分」這些字，其實是指「二分之一、四分之一、八分之一」的意思，是在說它們相對於全音符的長度。 這邊要注意的一點是，不同的音符，只有不同的「相對」長度的意義，我們並沒有為任何一種音符設定「絕對」的長度；我們只知道二分音符是全音符長度的一半，但我們並沒有設定全音符的長度是幾秒。 大部分的音樂老師都會跟音樂初學者說，四分音符的意思是一拍，雖然四分音符的確是最常被當作一拍的音符，但是其實你要設定哪種音符當作一拍都可以；你可以用之後我們會說到的拍號（time signature），來設定哪一種音符要被當作一拍。 其實我們還有比全音符更長一倍的倍全音符，和更多條尾巴的三十二分、六十四分、一百二十八分音符，不過我覺得它們都不是那麼常用。 這些音符都不常用

附點音符（Dotted Note）

你可以在音符的右方加上一個或多個小圓點，來改變音符的長度；假設有 n 個圓點，音符就會變成原來的「2 的 n 次方，分之，2 的 n+1 次方減 1」倍的長度。 附點音符的公式 假設有一個四分音符，被當作一拍，那麼： 右邊加一個點的話，就會變成 3/2 倍，也就是 1.5 拍； 兩個點的話變成 7/4 倍，也就是 1.75 拍； 三個點的話變成 15/8 倍，也就是 1.875 拍，依此類推。 每增加一個點，就是依序增加原音符時值的 1/2、1/4、1/8⋯⋯

連音（tuplet）

如果要把一個時間單位，平均分割成一般音符無法分割的長度，就要使用所謂的「連音」；最常見的連音，是 3:2 的和 5:4 的這兩種。 如果想要在一個四分音符的時間內平均彈三個音，因為沒有任何一種音符可以直接將四分音符切成三份，所以我就得要寫三個 8 分音符，然後標上數字「3」。 如果是要在一個四分音符的時間內，平均彈五個音，同樣的，沒有任何一種音符可以直接將四分音符切成五份，我就得寫五個 16 分音符，然後標上數字「5」。 三連音和五連音 不管是要幾連音，通常都是要用總時值大於且最接近你的目標長度的音符來記譜，並標示數字，依此類推。

關於音符的、一些拉哩拉雜的事

碎音（acciaccatura）：這種小字體、加上斜線的音符，意思是很快地彈過去，只占非常短的時間。 二度雙音：寫雙音的時候，如果之間的距離是二度（例如像 C 和 D），必須寫左右錯開，而且低音要寫在左邊。 音符各部位的名稱：你知道音符的各部位名稱叫做什麼嗎？圓圓的叫做符頭（note head），直線叫做符幹（note stem），尾巴就叫做符尾（note flag）；如果有連續兩個或以上的有符尾的音符，可以連在一起變成符槓（note beam）。 符幹和符尾的方向：如果音符是在第三線或以上的話，符幹要黏在音符的左邊並朝下，不然符幹就要黏在音符的右邊並朝上。注意如果音符有符尾的話，不管符幹是朝上還是朝下，符尾都一律在符幹右邊，很多初學者都會把方向寫反。 符幹和符尾的方向示範 雙聲部：在同一根符幹的音，拍子一定要一樣長；如果有兩個音要同時出來但持續的長度不同，就一定要把單行譜分成兩個聲部，一個聲部符幹往上，另一個往下。 雙聲部寫法

五線譜（five-line staff）

你一定知道五線譜有五線四間，線和間都可以放音符，音符的位置越高，代表的音高就越高；音高如果超過五條線涵蓋的範圍的話，你可以在上或下寫上加線（ledger line）。 要注意的是，五線譜只是在說位置越上面音高越高，這五條線本身，並沒有對應到特定音高的意思，我們需要用「譜號」來設定這些線和間代表什麼音高。

譜號（clef）

最常用的兩個譜號是 G 譜號（G clef）和 F 譜號（F clef）。 G 譜號：長得像是以下這樣子，這個符號的意義是，通過它肚子正中央的那一條線，要當作中央 C 往上的 G。 G 譜號最常被放在第二線，如果 G 譜號被放在第二線的話，我們就叫它「高音譜號」。 言下之意就是，G 譜號其實不一定要被放在第二線，我也可以把 G 譜號放在第一線，這樣的意思就是，我要把第一條線當做 G。不過現代樂譜裡，已經幾乎看不到 G 譜號在第二線以外的用法就是了，所以你不需要太擔心這個。 這份譜裡面全部的音都是 G F 譜號：邏輯跟前面一樣，意思是要把通過它的兩個點的那一條線，當做中央 C 往下的 F。 F 譜號最常被放在第四線，如果 F 譜號被放在第四線的話，我們就叫它「低音譜號」。但同樣地，F 譜號也可以被放在別條線，不管它放去哪裡，它兩點間夾住的那一條線都要當做低音 F。 這份譜裡面全部的音都是 F

更多關於譜號

G 譜號和 F 譜號在古早古早的時候，真的就是寫字母 G 和字母 F，以下是它們的演進過程（出自 Grove 字典）。 C、F、G 譜號的演進過程 C 譜號：彈鋼琴的人很少看到這個，意思是通過它正中央的那一條線要當作中央 C。C 譜號可以被放在任何一條線上（如下圖）。 C 譜號：這份譜全部的音都是中央 C 打擊樂譜號：如果你要記的是沒有明確音高的樂器，你要用這一種有兩條粗粗直線的打擊樂譜號。 高低八度：譜號的上下方可以寫數字 8 或 15，來表示整個譜號要高或低一個或兩個八度。

升降記號

先來說符號本身： 井字符號「♯」叫做升記號，意思是高半音，Unicode 號碼是 U+266F。 長得像小寫 b 的符號「♭」是降記號，意思是低半音，Unicode 號碼是 U+266D。 「♮」符號叫做本位記號，或口語更常講還原記號，是用來取消升記號或降記號的效果，Unicode 號碼是 U+266E。 低兩個半音的重降記號「 𝄫」就是寫成兩個降記號的樣子，Unicode 號碼是 U+1D12B。 高兩個半音的重升記號不是寫成「##」，而是寫成「𝄪」，我也不知道為什麼，Unicode 號碼是 U+1D12A。 五個常見的升降記號 我們用中文講話或寫文章的時候，我們習慣把升降記號說在或寫在音名的前面，譬如說「降 E」，「升 G」之類的；但注意如果你是用英文的話，升降記號要放在音名之後，像是「E-flat」和「G-sharp」才是正確的。 如果直接用符號寫的話，也是要遵循英文的寫法，也就是寫成「Eb」和「G#」，請不要再寫成「bE」和「#G」了喔，會讓你看起來很不專業。 還有一點是，並不一定是有升降記號的音，在鋼琴上就是黑鍵，升降記號的意思只是高低半音而已。例如說像是 E 這個音，升高半音的話變成 E# 也是一個白鍵；以及 C 降半音的話會變成 Cb，它也是一個白鍵。 有升降記號的音也可以是白鍵

臨時記號（accidental）

寫在樂譜小節裡面的升降記號叫做臨時記號，它的效用只有當下小節的同一個音而已。 如上圖，例如說我有一個 F，在第一拍被標了升記號，那麼這個第二拍同一間的 F，就還是要彈成 F#。但是第三拍這個高八度的 F，就不受到升記號影響了。 這種臨時記號只要跨過小節就會自動取消，所以像這樣下一小節還有 F 的話，就會自動變回還原 F，但如果覺得看譜的人會容易混淆，還是可以選擇加上提醒式的還原記號。 然後記得所有的臨時記號，都要寫在音符的正左邊同一線或同一間的位置。

調號（key signature）

寫在每一行最前面的升降記號是永久性的，它會影響整份譜任何一個八度的音；因為它有標記當下的音樂是在哪一個調的功能，所以我們把它叫做調號。 七個降記號和七個升記號的調號 在寫降記號的調號時，從左到右的順序是：Bb Eb Ab Db Gb Cb Fb 在寫升記號的調號時，從左到右的順序是：F# C# G# D# A# E# B#

拍號（time signature）

譜的最開頭上下並排的兩個數字叫做拍號，是用來設定拍子數量跟單位的符號；上面那個數字設定每小節有幾拍，下面數字設定每一拍用什麼符號寫。 如下圖，3/4 拍的意思就是： 一個小節有 3 拍 每一個拍子用四（4）分音符代表 兩個常用的簡寫如下圖： 這個長得很像英文字母 C 的東西，意思是 4/4 拍 中間有一條直線的 C，意思是 2/2 拍 有些樂理書裡面寫這個 C 符號的由來是因為英文的「Common Time」，這個是不正確的；其實它不是 C，而是一個半圓，是中古時期的定量記譜法（mensural notation）遺留下來的產物。 定量記譜法（mensural notation）

節拍器速度

譜最前面會出現的還有節拍器速度，例如像下圖「四分音符等於 120」的意思就是「一分鐘彈 120 個四分音符」。

反覆記號

在樂譜中的單直線叫做小節線，是用來標示拍子的循環；如果你要分段，或是要換調號，通常我們會寫雙小節線；另外那一種右邊的線比較粗的雙小節線叫做終止線，是表示整個樂章或曲子的結束。 帶有兩個點點的雙小節線，通常一左一右成對出現，表示它們的範圍內要演奏兩次，叫做反覆記號；如果左邊那個反覆記號沒出現的話，就表示「從頭到右邊反覆記號的範圍」要演奏兩次。 一些關於反覆的術語： D.C.：唸成 Da Capo，意思是從頭開始 D.S.：唸成 Dal Segno，意思是從「𝄋」符號開始 Fine：不是英文「我很好 I’m fine」的 fine ，而是義大利文的 fee-neh，表示結束的意思；Fine 通常跟 D.C. 或 D.S. 一起用，表示 D.C 或 D.S 之後，演奏到寫 Fine 的地方結束。 一號框框和二號框框：和反覆記號一起用，第一次演奏的時候要彈一號框框裡面的東西，第二次的時候則是要跳過一號框框的範圍，直接進到二號框框；它們的英文叫做 first ending 和 second ending，直接翻譯是「第一結尾」和「第二結尾」，但我們在台灣所有人都習慣講「一房」和「二房」，你如果講「第一結尾」和「第二結尾」會沒有人知道你在說什麼。 一號框框和二號框框 兩點中間夾一個斜線的符號，表示重複前一個小節。

弧線

弧線有兩種可能的意思，一種是連結線（tie），另一種是圓滑線（slur），線的本身看起來是一樣的。 如果弧線是連結剛好兩個同音，那麼它就是連結線，意思是把音持續住兩個音符加起來的長度；注意要同時滿足同音和剛好兩個音的條件才是連結線，不然它就是圓滑線。 連結線和圓滑現的區分 圓滑線是指範圍內的音要圓滑地演奏，基本上就是不要斷掉的意思，不過關於實際上圓滑奏要怎麼執行，就又是一個很長的話題了。 → 看這部影片

鋸齒線

image-20220618194305732 從一個音到另外一個音當中的斜斜的鋸齒線，指的是滑音（上圖第一小節） 寫在一個和弦旁邊垂直的鋸齒線，指的是琶音，預設是從低到高一個音一個音彈上去（上圖第二小節） 要設定琶音從高音往低音彈的話，要寫往下的箭頭（上圖第三小節） 在音的上下方標示數字 8 或是 15 的線，表示範圍內要移高或移低一或兩個八度（上圖第 4~6 小節） 帶有 tr 字樣的鋸齒線指的是顫音（trill），就是本音跟上方音快速交替，例如我有一個 C，上面有這條線的話，就是指 C 跟 D 快速交替；trill 可以從本音開始，也可以從上方音開始，至於哪一個情況該從哪裡開始，又是一個很長很學術的主題了。（上圖第一小節） 短的鋸齒線，在絕大部份情況的執行方式是「上本上本」，或「本上本」，以 C 為例的話，就是指快速彈 D C D C 或是 C D C（上圖第二小節）。 短的鋸齒線中間加一槓，意思是「本下本」，也就是 C B C。 S 型要彈「上本下本」或「本上本下本」，也就是 D C B C 或是 C D C B C。

Tremolo

震音（Tremolo）如上圖，意思是同音快速反覆，上方的斜線數代表反覆的頻率（兩條的話為 16 分音符、三條為 32 分音符），但有時候畫三條線也可以表示「很快」的意思，不見得一定要演奏成準確的 32 分音符。 如果是兩音快速交替反覆，可以在中間畫斜線，同樣的，斜線數代表反覆的頻率。
註：在我者小的時候，trill 是被翻譯成「震音」，而 tremolo 是「顫音」。但不知何時，台灣的教育部把兩個名詞的中文翻譯調換過來了，造成兩個世代溝通這個名詞的時候容易誤會，因此我建議你用這兩個名詞的時候，直接使用英文就不會有問題囉！

常見演奏法記號

一些音符正上方或正下方常有的記號： 圓點的意思是斷奏，就是把音彈得比記譜上短，跟後面的音中間有空白。 「>」符號意思是「強音」，也就是把那顆音彈比較重。 橫線記號，是把那個音拍子彈滿，或甚至是拖長一點點，有時候也有加強的意思。 延長記號，是在那個音停住，到你覺得爽了再繼續接下去的意思。

表情（強弱）記號

常見的強弱記號只會出現三種字而已，f 和 p 和 m，都是義大利文的縮寫；我相信大家應該都知道意思，但很多人不知道怎麼唸。 f 唸成 forte，意思是大聲。 p 唸成 piano，意思是小聲。沒錯，「鋼琴」這個字的原意就是小聲的意思，至於為什麼是這樣，請看這一部影片。 m 唸成 mezzo，意思是一半。 這三個字可以組合成六個等級，從最小聲到最大聲依序是 pp → p → mp → mf → f → ff，pp 要唸成 pianissimo，ff 要唸成 fortissimo。據我的理解，義大利文的 -issimo 字尾，應該是類似像是英文的 -est 字尾的最高級的意思吧？ fp 連在一起用的意思是大聲之後馬上小聲；sfz 唸成 sforzando，意思是突然比較大聲。 兩條斜線漸漸打開，意思是越來越大聲，漸漸關起來，意思就是越來越小聲，我們在中文講漸強和漸弱；除了用斜線表示，你也可以寫文字縮寫 cresc. 和 dim. 來表示漸強和漸弱。

鋼琴踏板

關於鋼琴踏板的記號 這個長得很像一隻狗的符號其實是文字「ped.」，意思是踩鋼琴的右踏板 「花符號」指的是放掉鋼琴的右踏板 不過我一直都不喜歡鋼琴譜上的這種踏板記號，因為這個符號只把鋼琴的踏板簡化成「踩跟放」兩種狀態，完全沒有考慮到深度的問題；所以我是覺得這個踏板記譜法應該要更新了，關於更多踏板的事情，請看好和弦超長篇的踏板文章喔！ una corda 字面上的意思是「一條弦」，是指踩鋼琴的左踏板的意思；因為古早古早時候的鋼琴，踩下左邊踏板後，鋼琴的琴槌會只有打到每個音的其中一條弦。 tre corde 字面上的意思是「三條弦」，也就是放掉鋼琴的左踏板，讓琴槌正常地敲到每個音的三條弦。

谱号

clef，(钥匙的意思)是五线谱中的一个符号，用来决定乐谱中不同位置表示的音高。谱号通常是一个音名字母的变体，放在五线谱中的某个位置上，表示该位置上的音高。谱号的作用范围到下一个谱号出现或者行尾。

三种类型的谱号

在现代音乐中，有三种谱号，分别是低音谱号（F）、中音谱号（C）和高音谱号（G）。每一种谱号表示了接下来的音符所在的位置代表了什么音高。
谱号 名称 标示音高 标示位置/准则 示例
高音谱号（G谱号） G4 调号中央的圈位
中音谱号（C谱号） C4 (亦称中音C/Middle C) 调号中央的位置
低音谱号（F谱号） F3 调号两点的中间位
当谱号的位置定了后，五线谱上不同位置的音高便能透过谱号所定立的基准而得出。同样，在单行乐谱阅读中，为免减少不必要的附加线或需要使用双行／多行式乐谱，在相关的位置更换谱号，便能更方便演奏者去读谱。

各款谱号可以根据实际需要，向上或向下调整它的标示位置，以达到最佳的显示效果，自谱号发明以来，音乐上曾出现过以下形式的谱号：
