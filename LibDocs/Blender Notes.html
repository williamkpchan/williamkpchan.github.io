Blender Quick Start Guide 10 minutes


selection: use right mouse button
deselecting: use the a key
toggle selection: use the a key
tumble view: middle mouse button, drag
pan view: shift + mmb
zoom view: ctrl + mmb
view selected: numpad period

Emulate Number Pad
go to file > user preferences (in the input section), 
click on the "Emulate Number Pad" checkbox which then allows you to use the regular period key to zoom to selected object.

move, translate, rotate object: use the object menu and arrows
move: g key
rotate: r key
scale: s key
properties: n key
tool shelf: t key
reset or set properties panel: select property or multiple properties and edit values

on object menu, different object modes available, imp is edit mode
edit mode: tab key
on object menu, different mesh selection modes available
mech select mode: ctrl + tab
snap menu: shift + s
on object menu, different pivot selection modes available
center cursor and view all: shift + c
