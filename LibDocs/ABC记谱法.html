ABC是什么

在 MIDI 领域，ABC 通常指 ABC 记谱法（ABC Notation），这是一种用纯文本表示音乐记谱的简化系统，与 MIDI 密切相关。

ABC 记谱法在 MIDI 中的角色




特性说明


本质纯文本音乐记谱格式，用字母 `A-G` 表示音高


与 MIDI 的关系可直接转换为 MIDI 文件（`.mid`）


优势人类可读、易于编写、文件极小、适合folk/传统音乐


转换工具`abc2midi`、`abcMIDI`、`EasyABC` 等





基本格式示例

```abc
X:1
T:小星星
M:4/4
L:1/4
K:C
C C G G | A A G2 | F F E E | D D C2 |
```

转换流程

```
ABC 文本→  abc2midi→  MIDI 文件→  播放器/DAW
```

ABC 记谱法语法指南


文件头（Header）


每首曲子以 X: 开头，常用字段：




字段含义示例


X:曲目编号（必须）X:1


T:标题T:小星星


M:拍号M:4/4


L:默认音符长度L:1/4（四分音符）


K:调号K:C 或 K:Am


Q:速度Q:120


C:作曲家C:莫扎特





音符表示




符号含义


C D E F G A B中音区音符


c d e f g a b高音区（小写高八度）


C, D,低音区（逗号降八度）


c' d'更高八度（单引号）


^C _C =C升、降、还原





时值与节奏




符号含义


C2双倍长度（二分音符）


C/2一半长度（八分音符）


C3/2附点四分音符


C>将下一音符时长匀一点给它（lengthen）


C<缩短，给下一音符（shorten）





小节与结构




符号含义


|小节线


||双小节线（段落结束）


|: :|重复记号


[1 ]1第一结尾


[2 ]2第二结尾

ABC 记谱范围

1. 基础旋律（你最常用）
- 调式：C、G、D、F、Am、Em… 所有大小调
- 拍号：4/4、3/4、6/8、2/4 等
- 速度：60–160 随便定
- 风格：流行、轻快、治愈、可爱、伤感、古风、日系、摇滚、电子
2. 带和弦的完整谱（可直接弹）
- 主旋律 + 和弦标记
- 流行进行：C–Am–F–G、15634145 等
- 适合吉他、钢琴、编曲
3. 多声部
- 主旋律 + 低音
- 主旋律 + 简单和声
4. 各种情绪风格
- 轻快阳光
- 温柔治愈
- 古风中国风
- 日系动漫
- 伤感抒情
- 可爱/儿歌
- 复古 80s 流行
5. 长度随便定
- 4 小节、8 小节、16 小节、完整一段主歌/副歌

你只要告诉我这 4 个信息，我就能直接出 ABC
1. 调式（如 C 大调）
2. 拍号（如 4/4）
3. 速度（如 120）
4. 风格/情绪（如：轻快流行、治愈、古风）

列出各种光的情绪风格

下面整理一套直接能用、好写进歌词 / 文案 / 旋律氛围的「光的情绪风格」，按情绪氛围分类，每类都配画面 + 感觉 + 适用场景，方便你写歌、写诗、写故事。


 一、温柔治愈系的光


晨光 / 破晓光
情绪：希望、新生、温柔、释然
画面：雾散、天微亮、城市刚醒
感觉：轻轻掀开一天，把难过照淡

暖光 / 夕阳光
情绪：安稳、怀旧、柔软、治愈
画面：橘黄、金粉、落在肩膀
感觉：被轻轻抱住，不刺眼，很安心

柔光 / 漫射光
情绪：平和、朦胧、温柔、舒缓
画面：云层滤过、窗边薄光
感觉：不尖锐、不张扬，像轻声说话

烛光 / 灯火微光
情绪：陪伴、坚守、微小却坚定
画面：夜里一点亮、摇晃却不灭
感觉：孤独里的依靠，黑暗里的锚点


 二、明亮积极系的光


正午强光 / 晴朗光
情绪：热烈、坦荡、清醒、力量
画面：天空湛蓝、万物清晰
感觉：干脆、明亮、没有阴影

星光 / 碎光
情绪：浪漫、梦幻、轻盈、向往
画面：点点闪烁、落在发梢
感觉：轻盈、干净、带一点童话感

曙光 / 希望之光
情绪：重生、坚持、破局、向前
画面：黑暗尽头一条亮线
感觉：终于等来，一切值得

追光 / 舞台光
情绪：耀眼、自信、被看见、绽放
画面：一束光打在身上
感觉：成为焦点，勇敢发光


 三、清冷孤独系的光


冷光 / 月光
