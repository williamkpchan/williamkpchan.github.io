Apache, Mercury, Tomcat server

Apache is the Apache Web Server also known as httpd. See http://httpd.apache.org
Mercury SMTP Server
Tomcat is a Servlet Container. See http://tomcat.apache.org

While at the simplest level both can be viewed as web servers that serve static files they have very different focuses.
httpd is typically used to host static content and dynamic content written in perl, php amongst others.
Tomcat is primarily intended to be used to serve web applications developed in Java to the Java Servlet specification.
