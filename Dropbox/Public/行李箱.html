揀行李箱

工欲善其事，必先利其器，幾個重點：

Check Point 1：行李箱的大小
行李箱通常可分為兩種，一種是登機箱，一種是需要托運的。登機箱多數是20吋或以下，而超過20吋的，普遍最常見的24、25吋箱，都必需要寄倉。留意乘搭廉航時，寄倉行李都需要附加費用。

Check Point 2：根據旅行日數而定
千萬不要打算一喼走天下，不同的旅程安排應該帶上不同的行李箱。3天短線遊，帶20吋的小行李箱應該足夠，而5-7日就可以使用的24、25吋的，如果是留學生或長程旅行，建議選擇27-30吋的大行李箱。當然，你去3天，但會瘋狂購物又是另一回事。

Check Point 3：軟硬度
硬喼保護性比較強，經得起撞擊，能防水，款式比較百變，不過會比較重。而軟喼則比較輕，且容量具彈性，因此可以「盡塞」，但防水性則比較差。

Check Point 4：選用之物料
比較防撞：聚丙烯(PP)
比較輕：聚碳酸酯(PC)
比較硬：ABS樹脂
比較彈性：尼龍/帆布

Check Point 5：輪子
四個輪會比兩個輪的較能承載重物，而360度轉動的輪子可以任意推動，相對較方便。

Check Point 6：拉桿
除了要留意拉桿的堅固度，也要留意其長度，因為合手的拉桿，會比較省力。
