增強記憶力的訓練

我們的教育制度，基本上十分著重學生們的記憶能力。

所謂好成績的學生，就是要他能把所讀的資料，好好地存放在腦中，並在測驗和考試之時，能把有關的資料，準確地拿取出來。

但到底記憶力是天生的？

還是可以後天培養呢？

 

記憶 = 儲存 + 运用

memory = storage + access

當然我們相信每個人都有不同的記憶力。

有些人對數字記憶甚佳，亦有些人對語言的記憶能力強。

我唸中學時的校長張子江先生，便能把全校學生的名字 ( 包括已畢業的同學 ) ，全都記在腦中。

可是對於那些沒有這種「記憶天份」的同學，是否就無法培養出良好的記憶呢？

相信很多同學都曾經有過這樣的經驗，就是在考試或測驗期間，突然忘記了某題目的答案，雖然自己十分清楚是曾經讀過，甚至連是在書本上的哪個位置也記得，可是答案總是在唇邊，說不出來；

而最不值的，就是當我們交了試卷以後，答案才徐徐地在腦海中浮現出來，可是卻為時已晚，這並不是我們真正的忘記了答案，而是未能有效地把答案找出來。

真正良好的記憶力，是不但能好好地把資料儲存，且要能有效地把資料回憶出來。

很多人常常說自己的記性不好，其實並非記性差，只不過是未能有效地回憶出來而已。

 

自然記憶 VS 加工記憶

自古以來，我們都有不同的方法來加強我們的記憶能力，但直至最近幾年，人類對大腦的運作與認識多了很多，故能夠發展一些較有系統的方法來加強我們的記憶。

其實發展記憶能力可分為兩大方向，一種是「自然記憶」 (NaturalMemory) ，就是我們平常用腦時所給與我們的記憶功能；

另一種是「加工記憶」 (Mnemonics) ，亦有人稱之為超級記憶法，就是我們用一些技巧，把要記的東西，刻意地留下一些記憶痕跡，好使在回憶的時候，能更有效的把需要的資料拿取出來。

兩種發展記憶的能力都是互為影響。

 

有效加強自然記憶的方法

大腦的運作與功能，是需要靠充份的休息來維持。

故足夠的睡眠是加強記憶必備的條件。

平均常人所需要的睡眠時間為每天八至九小時，但真正能夠幫助記憶的，是睡眠的「質」，而不是「量」。

若要睡得好，在就寢前不要進餐，或是過飢，而在冬天時，最好是先用熱水把雙腳暖和，然後才上床入睡，這都是幫助我們進入優質睡眠的條件。

 

此外，大腦在平日的運作中，也需要很多中段休息時間，故在每次溫習功課及讀書時間，最好是每半小時後有兩三分鐘的小息。

這些小息不但能活化我們的腦筋，而且能製更強的記憶效應，因大腦是能較容易把學習時前段和尾段的東西記憶起來。

把學習分成很多的小段便是加強這前段尾段的記憶效應。

 

大腦的最佳補品

記得以前在唸書時，每逢考試母親都喜愛燉豬腦來幫助我們補補腦，這都是中國傳統以形補形的方法。

又有些家長喜歡要孩子飲雞湯，食雞精，以及維他命等等東西。

目的都是希望幫助孩子建立更好的頭腦。

其實人類大腦的最好補品、是空氣和水，這是上帝造人時給與人類最大的禮物。

人人都知道大腦是不斷地需要新鮮空氣 ( 氧氣 ) 來維持其功能和運作，如氧份不足，其功能也會顯著減退，更甚者便是對大腦造成傷害，甚至死亡。

故此，吸煙和飲酒都是大腦的最大敵人，因二者都會大大消耗人體血液內的氧份而使大腦缺氧。

(I) 鍛鍊呼吸有助記憶

另外，呼吸雖是人類天生的本能，沒有呼吸便沒有氧氣，一旦缺氧，人便不能再繼續生存；

可是並不是人人都曉得正確的呼吸方法。

所謂正確的呼吸方法，首先在我們做深呼吸時，當吸入大大的一口氣時，小腹是徐徐擴大，而在呼氣時，小腹卻是漸漸收縮。

這亦是學唱歌時的腹式呼吸，和學中國武術的丹田氣。

故在最初練習時不妨慢慢地一吸一呼，而同時小腹是一脹一縮地加以配合。

當習慣了這一呼吸方法時，便可嘗試運用「吸四、停四、呼四、停盡」的方法。

就是用約四秒的時間吸氣，停四秒鐘，再用四秒的時間呼氣，再停四秒，這謂之一呼吸循環，同學可在每天早上起床時做八至十次這種循環呼吸，然後在每次讀書前再做三至五次，便能提升我們大腦的記憶能力。

(II) 清水能提高記憶能力

清水亦是人類大腦最重要的原素之一，因人類的大腦中有百分之七十以上是水，而大腦中細胞與細胞的資料傳遞及運作功能，都是用水作為傳送的媒介，故此足夠的水份是令大腦運作最佳的必須條件。

可是很多人並不經常留意自己的身體是否有充足的水份，一個普通人每天最少要喝六至八杯清水，所以要養成每天勤喝清水的習慣，並且在每次讀書或溫習前，都必事先飲杯清水，便能提高自己的記憶能力。

 

壓力是發揮人類潛能的最大阻礙

壓力是發揮人類潛能的最大阻礙，更是最能使我們大腦功能癱瘓的最有效武器。

故盡可能為子女們製做一個無壓力的學習環境，就如兩三歲的小孩子學習一樣，他們沒有壓力，亦不曉得甚麼是壓力，所以小孩子們學習是最多最快，而且記憶是最悠久的。

不但如此，如果能夠幫助他們保持一個積極愉快的心情，更大大有助提升他們記憶潛能。

若要能達到積極愉快的心理狀態，我們可經常鼓勵孩子和協助他們回憶過往愉快學習和成功記憶的經驗。

記憶力對任何類型的學習，都佔著一個十分重要的地位，尤其是現今教育制度與考試模式，大部份基本是對同學記憶能力的測試，能掌握良好記憶技巧者，便是優良學生，否則又很難在這制度上有出頭的一日。

可惜，絕大部分的學校，從來沒有教導同學有效的記憶技巧，老師們上課時，又沒有教導學生如何把課堂上所教科目的重點記下來，因為一直以來，要把課文牢牢地記熟是學生們讀書該付的責任，考試時未能把正確答案寫出來，便是學生沒有盡上努力讀書的本份

 

記憶力需要鍛鍊

倘若老師能在他們教學時多行一步，接受多一份額外的挑戰，就是在每次上課時，不但只是教導學生們了解明白課文的內容，而且能夠進一步地幫助同學們運用一些有效的方法和技巧，把所教的重點，幫助學生在下課前時能完全記憶下來，這樣，同學們的學習不是能事半功倍嗎？

他們對學習不是能更有信心和把握嗎？

上課讀書不是能變成肯定他們才幹能力的機會，而再不是不斷繼續為他們製造失敗的經驗嗎？

記憶力並不是學生們天賦的本能，它與任何其他技能一樣，是需要別人的指點、教導，然後自己加以練習而得。

 

到底我們要如何才能即時有效地加強我們的記憶呢？

首先我們要知道人是如何把資料儲存在腦海中。

當我們要記憶一些資料或課文時，到底我們是如何去把它們牢牢地「記」下來的呢？

 

圖像式記憶

其實大部分人，都是用圖像的形式，把資料儲存起來；

而大部份人的思考過程，都是用圖像來處理，而絕非用文字，文字只不過是用來把我們思想的意念，能清晰具體地表達出來的工具。

故所以絕大部分人類早期用作記事的方式，全都以圖畫來表達。

其實我們大家亦可作一個簡單的測試，當我們聽見「蘋果」二字時，在我們的腦海中會有甚麼反應的呢？

你是在腦海中看見「蘋果」兩個中文字，抑或是看見一個具體的「蘋果」，甚至連它的顏色也可以見到。

相信世界上絕大部分的人都是看見一個「真實的蘋果」，因為人類從五官所接收的資料和知識，絕大部份都是藉視覺渠道所接收的。

 

發揮幻想力

當我們希望有效地把一連串資料放在腦海時，把那些資料開聲朗讀幾遍，或是抄寫幾次，並不是幫助我們記憶的最佳方法，而是要運用我們的幻想力，亦即是我們常說的右腦潛能，把要記的資料存在腦海中，一一用圖像呈現出來。

上帝創造人類，給與人類其中一項最重要的本能，便是有這樣一種「幻想力」，這「幻想力」亦是人類潛能蘊藏之所在，它不但把人類與其他動物區別開來，而且更超越了其他動物。

 


所以當我們要把課文重點記下來時，最主要的記憶方法，是運用我們的幻想力，在我們的腦海中用圖像化的形式展示出來。

當我們的幻想力越能夠清楚，我們所幻想的畫面越是夠大，我們的記憶便越深刻。

不但如此，如果我們能在幻想圖像化的過程中，多加另外的一些感官及情緒時，即是多加點聲音、味道、氣味、觸覺和喜怒哀樂等情緒，我們便能更牢固地把想要記住的東西儲存在腦海中。

 

架疊法記憶

例如我們要記下中國東北三省的一些重要工業，有石油、鋼鐵、煤炭、森林、造紙、化學、機電、機械和汽車等，最基本的記憶方法，就是把這些東西逐一地活現眼前，並以一件物件架疊在另一件事物上的「架疊法」，來把重點串連起來。

即首先我們要在腦海中幻想出一桶桶的石油，最好甚至連石油的氣味，它給你的感覺等都帶進幻想裡去；

然後在石油的上面，我們看見正在火紅紅中提煉的鋼鐵，最好連那在煉鋼爐中熾熱的感覺也能幻想出來；

跟著放在這鋼鐵上的，正是一些黑得令人討厭的煤炭，如是者一樣東西疊在另一樣物件上，於是我們便能快而有效把東西記進去。

但是請注意，在整個記憶的過程中，都必須是用幻想力在腦海中清清楚楚地展示出來。

 

幫助孩子背書亦可用這個方法，就是把所要背的課文內容，如電影般一幕幕地在腦海中演繹出來，這都會大大加強我們背書的速度和效果。

 這個方法在初時運用，可能不會太習慣，但大部分小孩子的幻想力都是十分強的，若能多些練習，必定能大大倍增他們的記憶力。

增強記憶力對學習的重要是絕對不容置疑的，然而記憶力究竟是天生的，還是可以後天培養的呢？

從我們的經驗所知，記憶力絕對是可以透過後天訓練來加強的。

以下是一些訓練記憶能力的方法。

 

增強記憶的方法

大腦攝影技巧

這方法其實很簡單，預先準備一些雜誌廣告，或舊相片，然後給孩子望三至五秒，嘗試要求孩子用影相的方法，把整個圖片攝錄在自己腦海中。

你可向孩子提問一些有關圖片內的問題，以刺激他們回憶起圖片的內容。

當孩子對這些遊戲感到純熟或容易掌握時，可把望圖片的時間逐漸縮減，直至只用一秒時間。

多練習回憶

記憶力是包括接收、儲存和提取資料三個過程。

要有效地在腦海中提取有關資料，就必須多作回憶力的訓練。

嘗試要求孩子把學校生活或上課時的情節告訴你，並追問一些細節或仔細的問題。

閒時翻閱一些家庭旅行或其他活動的照片，然後請孩子回憶當時的情況。

甚至在逛公司時，行過一些櫥窗後，回憶櫥窗內的東西。

這些都是可以閒時當作遊戲的練習。

大腦電影調校練習

嘗試在大腦中製造不同事物的影像，並且把腦海中的螢光幕作出不同的調校，例如把螢光幕的畫面放大，音量擴大，甚至改變它們的色彩、形狀、位置、光暗、遠近等，都是對記憶力大有幫助的練習。

白日夢

很多家長對孩子發白日夢抱負面的態度，認為這是浪費時間的玩意，但其實發白日夢對發展右腦記憶功能有很大的幫助。

你可以每天給予孩子十至十五分鐘自由自在地幻想，並為他們提供一處安靜而不被騷擾的地方。

繒圖

多用色彩繪圖，把自己腦海所想的事物繪畫出來。

這是有助加強右腦思維和記憶的訓練。


以上方法，若要在孩子身上有效地實行，就必須把這些練習當作遊戲，讓孩子能在歡樂中學習，孩子們的記憶能力便會漸漸提昇。

 

方法

做事情都有方法，學習有方法，讀書有方法。

而如何加強我們的學習效果，就是要增強記憶力，因此增加記憶力也有方法，以下即介紹一些增強記憶力的方法。



第一、吸收足夠的營養，記得補充能夠幫助我們記憶力、提升思考力和腦力，以增加平衡感的食物。



第二、多運動，尤其是有氧運動，因為它可以幫助我們增加腦內的氧氣，可以讓腦細胞活化，提升記憶力。



第三、多思考，例如利用拼圖、拼字、猜謎等方法刺激腦細胞，就能增強記憶力。



第四、讓自己放輕鬆，可以減少更多的壓力，也可以幫助我們集中精神。



第五、增強自己對記憶力的信心，要相信自己一定可以記得很好，只要我們能夠運用好的記憶策略及原則。



第六、運用增強記憶的方法，因為透過增強記憶的方法，可以幫我們記憶的更深、更長、更有效。

　　

頭尾記憶法：何謂頭尾？

即將材料加以分段，每一段學習十分鐘～十五分鐘，每一段就有它的頭尾，如此我們對開頭與結尾都會記得特別清楚。




比較記憶法：例如將新的知識和舊的知識加以比較。



列表方法：例如將內容知識轉化為數字加以統計，即一個統計的圖表法，再則就是系統法，也就是將學科知識，依其邏輯順序加以安排系統化。



睡眠記憶法：例如安排在早上起床時學習，此時是記憶力最清晰的時刻，亦可以在睡前安排一段我們喜歡的或是覺得比較有難度的東西來學習，因為這樣入睡之後就不會受到干擾。

當然也可以在睡眠的時候，可以播放學習的錄音帶，也可以透過我們潛意識意識來學習，這樣的學習效果是非常好的。



自我測驗法：當我們學了一段之後，便要檢測自己學習的成效如何？

自問自答，這也是可以自我測驗的方法。



其它：例如組織法，將所學過的內容加以重新安排、整理，這樣的組織，一定讓我們記得很長、很久，另外還有聯想法、關鍵字法、故事法等。



　　　　以上我們所提到了這些強化記憶之方法，希望各位能夠好好地再加以不斷的運用，用得越多，就越純熟，自己也能變成行家。
